## Development
- pull repository
```sh
git init
git remote add origin git@github.com:imc-ux/AdvancedReact.git
git pull origin main
```
- dependencies installation
```sh
npm install
```
- local testing
```sh
npm run server
```
- push your changes
```sh
git init
git add file
git commit -m "modify information"
git branch -M main
git remote add origin git@github.com:imc-ux/AdvancedReact.git
git push -u origin main
```
